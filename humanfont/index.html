Humanitarian Font

The iconic font designed for the
Humanitarian

Download the
Font View Project
on GitHub

All the icons you need in One font

Humanitarian Font is a pictographic language of humanitarian & web-related actions. The font has been generated with FontForge.

CSS control

Easily style icon color, size, shadow, and anything that's possible with CSS. Check some examples here or here.

Infinite scalability

Scalable vector graphics means every icon looks perfect at any size and looks ok on high-resolution displays like retinas screen. Compared to PNG icons, it will better render and can be used in GIS software like ArcGIS.

Free to use

Humanitarian font is based on other fonts like the Awesome Font. As such, the same licensing terms are used.

Designer Friendly

Humanitarian font is great for infographics designers! Install humanitarian-webfont.otf and simply copy & paste page.

Open to suggestion

Share feedback and comments on the Humanitarian Font GitHub project.
